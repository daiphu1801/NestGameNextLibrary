# Backend Architecture

## Tech Stack
- **Framework**: Spring Boot 3.2
- **Language**: Java 17
- **Database**: PostgreSQL
- **Security**: Spring Security + JWT
- **Build Tool**: Maven

## Layers
1. **Controller**: REST Endpoints
2. **Service**: Business Logic
3. **Repository**: Data Access (Spring Data JPA)
4. **Entity**: Database Tables

## Key Features
- JWT Authentication (Login, Register, Refresh)
- Game Management (Search, Filter, Pagination)
- User Profile (Favorites, Play History)
